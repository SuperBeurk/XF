Detailed Description

Base class for all types of events and timeouts.

Requirements:

  • Needs to hold a property which allows to distinguish between different types of events (eventType_).
  • Use an enumeration to define the type for the event type property (XFEventType).
  • The event type property needs to be set at initialization time and cannot be changed afterwards.
